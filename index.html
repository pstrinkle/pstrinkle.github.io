jQuery Plugins

The following are some open source jQuery libraries I wrote with varying degrees of utility.

  • js-ngrams - Given English as a training language, randomly generate words or names.
  • js-modeling - Fun implementation of bigrams and trigram generative models as well as vectorspace text analysis.
  • jquery-facebook-authorize - Simplified way of requesting Facebook permissions that determines whether your minimum set of required permissions are granted before claiming success
  • jquery-levelup - Basic +1/-1 text animation
  • jquery-follow-cursor - Basic ability to rotate any element(s) to follow the mouse/cursor
  • jquery-paintbox - MS Paint emulator
  • jquery-snake-game - Classic snake game extending paintbox :D
  • jquery-proximity-glow - DOM elements glow based on how close the user's mouse gets
